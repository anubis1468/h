# SurgeCoin Press Kit

## Project Summary
SurgeCoin is a hybrid crypto ecosystem that merges virtual-world mining with real GPU mining, enabling gamers and miners to earn in a shared, player-owned economy.

## Key Facts
- Dual-mining model: virtual + GPU
- Two tokens: SCN and SCN-G
- Unified wallet and marketplace
- Hybrid tournaments and events

## Boilerplate
SurgeCoin is redefining digital value by combining gaming, mining, and decentralized finance into a single, accessible ecosystem.
